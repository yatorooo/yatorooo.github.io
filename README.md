# Teke Xu Personal Website

Static personal site for GitHub Pages.

## Files

- `index.html`: page structure and content
- `styles.css`: visual design and responsive layout
- `script.js`: sticky header and active navigation state

## Repository

- GitHub Pages repo: `https://github.com/yatorooo/yatorooo.github.io`

## Publish On GitHub Pages

1. Commit these files locally.
2. Push to the `master` branch of `yatorooo/yatorooo.github.io`.
3. In GitHub, open `Settings` -> `Pages`.
4. Set the source to `Deploy from a branch`.
5. Choose the `master` branch and `/ (root)` folder, then save.

GitHub Pages will publish `index.html` automatically.

## Content To Update
